# Test-Management-Jira-Zephyr
This repository showcases my hands-on learning and practical experience with test management using Jira integrated with Zephyr. It includes test case creation, execution, traceability, and report generation.
**What I Learned**
Creating test cases directly in Jira (Zephyr)
Importing test cases from Excel into Zephyr
Creating and managing test cycles
Adding test cases to test cycles
Executing test cases within a test cycle
Tracking test execution status
Maintaining requirement traceability
Generating test execution reports
**Features Demonstrated**
✔️ Test Case Management
Defined test case name, objective, and preconditions
Structured and organized test cases
✔️ Test Case Import
Bulk import of test cases using Excel
✔️ Test Cycle Management
Created test cycles
Assigned test cases to cycles
✔️ Test Execution
Executed test cases step-by-step
Updated execution status (Pass/Fail)
✔️ Traceability
Linked test cases with requirements for tracking
✔️ Reporting
Generated test reports and analyzed results
